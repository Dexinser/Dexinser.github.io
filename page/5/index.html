Vue

Vue：

总结一下使用vue的一些方法吧~

用vue-cli来创建一个vue项目的过程：
首先在命令行工具页面输入：vue init <template-name> [project-name];之后按提示进行就行了。
创建成功脚手架之后打开创建成功的文件夹，在所在的文件夹下面：cnpm install 安装开发项目所需要的所有依赖；
npm run dev 运行本地测试开发开一个本地服务器。
